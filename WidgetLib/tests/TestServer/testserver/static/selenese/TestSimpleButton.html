TestSimpleButton
tests/static/TestSimpleButton.html: This test suite checks basic rendering of a simple button, and in particular that the values supplied by the application controller are reflected in the rendered page (the test controller in turn gets these values from the URI).
Works in conjunction with templates, TestSimpleButton.kid, which itself extends TestMaster.kid.
open http://localhost:8080/Tests/TestSimpleButton  
assertTitle Simple button  
verifyAttribute name=sbname@value A simple button
verifyText name=sbname
open http://localhost:8080/Tests/TestSimpleButton?name=button1  
assertTitle Simple button  
verifyAttribute name=button1@value A simple button
verifyText name=button1
